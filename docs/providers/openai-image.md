# OpenAI Image

- Driver: `openai_image`
- Recommended model: `gpt-image-1.5`
- Auth: `OPENAI_API_KEY`
- Default response format: `b64_json`
- DALL·E 2 and DALL·E 3 are deprecated by OpenAI and documented as supported only until May 12, 2026.

LoreForge downloads or materializes the image locally before persisting the episode, so it does not depend on temporary OpenAI URLs after generation.
